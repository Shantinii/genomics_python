Throughout my master's there were few projects that I had to build along with my teammates. Here are some of the codes for the projects I have done 😊
## 1. Palindrome detection system 
This is a python based palindrome detection system that identifies spaced and perfect palindrome in given DNA nucleotide sequences. The sequences can be manually inputed or via a FASTA file (100 bp). The program validates nucleotide sequences, generates reverse complements, and displays detected palindrome regions with positional information.
